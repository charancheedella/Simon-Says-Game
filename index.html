Simon Says Game

Instructions

1. Press the Start button.
2. Watch the colors blink in a pattern.
3. Remember the order of colors.
4. Then, Click the colors in the same order.
5. Each level adds one more blink at the end.
6. If you get one wrong, the game ends!

Click on Start button below to start the game!
